```typescript
interface DownloadedVersionState {
  version: string;       // semver e.g. "0.2.0"
  releaseTag: string;    // e.g. "v0.2.0-dev"
  sha256: string;        // hex SHA-256 of the binary file
  downloadedAt: string;  // ISO 8601 e.g. "2026-04-01T12:00:00.000Z"
  binaryName: string;    // e.g. "hand-x-darwin-arm64"
}
```

Desktop resolves binary via resolveHandXBinaryPath():
1. Checks ~/Library/Application Support/gh-desktop-app/bin/hand-x-darwin-arm64 exists
2. Reads hand-x-downloaded-version.json, parses as DownloadedVersionState
3. Compares downloaded version against bundled version (downloaded wins if >= bundled)
4. Verifies SHA-256: reads binary bytes, computes sha256, compares against state.sha256
5. If mismatch: logs "SECURITY: Downloaded binary SHA-256 mismatch!" and falls through to bundled
